- `bg_room_night.png` - Phòng ngủ ban đêm
- `bg_room_day.png` - Phòng ngủ ban ngày
- `bg_school_class.png` - Lớp học
- `bg_canteen.png` - Canteen/cafeteria
- `bg_cafe.png` - Quán cà phê
- `bg_alley.png` - Ngõ hẻm
- `bg_stage.png` - Sân khấu thuyết trình

## Audio

### Background Music (BGM)

Place these files in `public/assets/audio/bgm/`:

- `bgm_calm_room.mp3` - Nhạc nền phòng yên tĩnh
- `bgm_tension_chat.mp3` - Nhạc căng thẳng khi chat
- `bgm_hurt_silence.mp3` - Im lặng đau đớn (gần như im lặng)
- `bgm_hope_practice.mp3` - Nhạc hy vọng khi thực hành
- `bgm_stage_resolve.mp3` - Nhạc quyết tâm trên sân khấu

### Sound Effects (SFX)

Place these files in `public/assets/audio/sfx/`:

- `sfx_phone_vibrate.mp3` - Rung điện thoại
- `sfx_keyboard.mp3` - Tiếng gõ bàn phím
- `sfx_message_pop.mp3` - Tiếng tin nhắn
- `sfx_rain_window.mp3` - Mưa ngoài cửa sổ
- `sfx_door_close.mp3` - Đóng cửa
- `sfx_crowd_murmur.mp3` - Tiếng đám đông

## UI Assets (Optional)

Place these files in `public/assets/ui/`:

- Phone notification frame
- Chat bubbles (AI + group)
- Journal paper texture
- Timeline/day badge

## Notes

- All images should be in PNG format
- All audio should be in MP3 format (or OGG for better compression)
- Recommended image size: 1920x1080 for backgrounds
- Recommended audio bitrate: 128kbps for BGM, 64kbps for SFX
